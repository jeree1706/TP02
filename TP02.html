1.1- ¿qué es una URL?

Una URL es el texto que indica donde se puede encontrar un recurso (ya sea una imagen, un video, pàgina, etc)

1.2- ¿qué es una url absoluta?

Es la direcciòn WEB que incluye el protocolo, el nombre de dominio y la ruta completa del archivo

1.3- ¿qué es una url relativa?

La Url relativa solo incluye la ruta de ubicaciòn del recurso. Es màs ràpida de codificar y un tiempo de carga ràpido

1.4- ¿cuál es la diferencia entre una url relativa y una url absoluta?

La diferencia entre una url absoluta y una url relativa es que las URL absolutas incluyen todas las partes de la URL (protocolo, servidor y ruta) y la url relativa es màs corta ya que se quitan partes. AL estar incompletas, es necesario disponer de información adicional para obtener el recurso enlazado.

1.6- ¿qué es un directorio? ¿qué relación hay con una carpeta?

Un directorio es un archivo que tiene la capacidad de almacenar archivos y se pueden crear subcarpetas. La relaciòn con la carpeta es que son sinonimos, ya que ambos pueden almacenar archivos

1.7- En la estructura de un proyecto web, ¿a qué se llama directorio raíz? ¿Con qué símbolo se representa?

Se llama directorio raiz al primer directorio o carpeta y aparece en la parte superior de un diagrama en árbol de un sistema de archivos. Se representa con el simbolo "/"

1.10- ¿Cuál es el propósito de la etiqueta anchor?

El proposito de la etiqueta anchor es crear un hipervìnculo a otra pàgina web, archivo o URL

1.11- ¿Qué atributo se utiliza en la etiqueta anchor para especificar la URL a la que debe enlazar?

El atributo que se utiliza en la etiqueta anchor es "href" para especificar la URL

1.12- ¿Qué sucede si el atributo "href" en la etiqueta anchor está vacío?

Si el atributo "href" està vacio, al hacer click no te lleva a ningun lado

1.13- ¿Cuál es el uso del atributo "target" en la etiqueta anchor y cuáles son los posibles valores que puede tener?

El uso del atributo "target" especifica dónde mostrar el URL seleccionado, permite que se abra una nueva pestaña.

1.14- Entonces ¿qué es un hipervínculo?

Un hipervinculo es un enlace que hace referencia a otro recurso
