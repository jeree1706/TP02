1.1- ¿qué es una URL?

Una URL es el texto que indica donde se puede encontrar un recurso (ya sea una imagen, un video, pàgina, etc)

1.2- ¿qué es una url absoluta?

Es la direcciòn WEB que incluye el protocolo, el nombre de dominio y la ruta completa del archivo

1.3- ¿qué es una url relativa?

La Url relativa solo incluye la ruta de ubicaciòn del recurso. Es màs ràpida de codificar y un tiempo de carga ràpido

1.4- ¿cuál es la diferencia entre una url relativa y una url absoluta?

La diferencia entre una url absoluta y una url relativa es que las URL absolutas incluyen todas las partes de la URL (protocolo, servidor y ruta) y la url relativa es màs corta ya que se quitan partes. AL estar incompletas, es necesario disponer de información adicional para obtener el recurso enlazado.
